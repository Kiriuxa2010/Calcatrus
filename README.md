# Installation
On Linux/Mac: 
1. Install Calcatrus Source Code
2. Install G++
3. Compile Source code: "g++ main.cpp"

On Windows:
1. Install Source Code
2. Install G++
3. Compile Source Code: "g++ main.cpp"
